## 🧩 About

**LoL Hack** is a collection of utilities for League of Legends that automates repetitive tasks and enhances gameplay. Based on popular open-source projects, it includes auto-accept, auto-pick, auto-ban, skin changer, zoom hack, and champion selection automation.

Key technologies used:
- **LCU API** — local client communication
- **Memory injection** — for skin changing and zoom hack
- **Python/Node.js** — for automation and custom scripts

## ❓ FAQ

**Is this detectable?**
Any third-party tool carries risk. Riot actively bans accounts using unauthorized modifications. Use at your own discretion.

**What is the LCU API?**
The League Client API is an undocumented local interface used by many legit automation tools. It's not officially supported and may change with updates.

**Can I use the skin changer in ranked?**
Skin changers modify client-side memory. Use with extreme caution — Riot detects memory modifications.

**Does the zoom hack work with all champions?**
Zoom hack works universally, but some champion abilities may revert zoom (e.g., Jhin, Xerath, Zoe R).

**Is this malware?**
No — but antivirus may flag it as a false positive. Download only from the official source.

**Does the tool need updates?**
Yes. Game patches change offsets and LCU API endpoints. Check the release notes before use.
